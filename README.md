# Order App (Phase 1)

Simple mobile-first ordering app for wholesale customers.

Phase 1 replaces handwritten order sheets with a clearer digital flow:
- Select products
- Enter quantities
- Review and save draft

## Tech Stack

- Vue 3 + TypeScript
- PrimeVue
- Pinia
- Axios
- Tailwind CSS
- Zod
- Vue I18n (English/Spanish)

## Current Features

- Bilingual UI (Spanish/English)
- Product catalog with image cards
- Clear selected-item indicators
- Quantity screen with large, readable controls
- Draft order persistence in local storage
- Auto dark-mode support from system preference

## Run Locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Goal

Make order entry faster, easier, and less error-prone for non-technical users in wholesale and retail environments.
